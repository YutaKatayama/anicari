# anicari

[AniCari](https://corp.petokoto.com/) 風デザインで構築する、AniCari コーポレートサイトです。

## Tech Stack

- **Next.js** (App Router)
- **Vercel** (デプロイ)
- **Supabase** (バックエンド / データベース)
- **Tailwind CSS**

## Getting Started

```bash
npm install
cp .env.local.example .env.local
npm run dev
```

[http://localhost:3001](http://localhost:3001) を開いて確認できます。
